To use this School Management Project you need to follow few steps:

	Step 1: Open Java eclipse
	Step 2: Go to File and Open Project from file system
	Step 3: Select the Project
	Step 4: Open MySQL Workbench create server localhost:3306
	Step 5: Import "schoolmanagement.sql" file into your local host and then do modification as per your choice

Note:-
	You don't need to do anything in this project it will be run but its just for learning purpose you can create your own according to this and make your school or college projects.
